Grouptabs

Track expenses in a group of people!

Share things with friends, have one person pay, and note it in the app. Next time someone else pays. You always know who is currently in red or green.

You can enter any constellation of participants for an expense. And have multiple people pay arbitrary parts of it. It’s easy and flexible.

Try the App

It is an offline-capable, installable web app and allows collaborating without user accounts. It works great on mobile devices!

You can contribute to the open source code and report issues & suggest features on GitHub. Also check out concepts, ideas and documentation.

Find the source code of the former version which used remoteStorage in the master branch. There’s an importer app to move your tabs from a Remote Storage to the new app.
